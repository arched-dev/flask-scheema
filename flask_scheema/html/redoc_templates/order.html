## Order By The `orderby` query parameter allows you to sort the API results based on one or multiple fields. You can sort in ascending or descending order. ### Syntax - **Ascending Order**: `orderby=field` - **Descending Order**: `orderby=-field` - **With Table Name**: `orderby=table.field` or `orderby=-table.field` > > ### Example > > `{{ base_columns[0] }}` in ascending order: `?orderby={{ base_columns[0] }}` > > `{{ base_table }}` on `{{ base_columns[1] }}` in descending order: `?orderby=-{{ base_table }}.{{ base_columns[1] }}` >
